                              # Vuex

                              • 官方解释 vuex是一个专为vuejs应用程序开发的状态管理模式

                              # 状态管理到底是什么？

                              • 状态管理模式。集中式存储管理区这些名词听起来就非常高大上，让人琢磨不透
                              • 其实，你可以简答的将其看成把需要多个组件共享的变量全部存储到一个对象里面
                              • 然后将这个对象放在顶层的vue实例中，让其他组件可以使用
                              • 那么多个组件是不是就可以共享这个对象中所有的变量属性了呢？

                              # state单一状态树的理解

                              vuex提供使用单一状态树，什么是单一状态树呢？

                              • 我们有很多个人信息需要记录，分布在不同系统内，如果你要办某个业务，你会发现提交很多信息，每个地方都要去盖章，跑流程，反之一个系统包含所有的信息，就是单一状态树的概念

                              //store/idnex
const mouduleA = {
  state: {
    name: "zhangsan"
  },
  mutations: {
    updatename(state, payload) {
      state.name = payload;
    }
  }
}
//const store里面
 modules: {
    a: mouduleA
  }
export default store
//app内
    <h2>---------------------App内容：module里面的内容</h2>
