      # || 或

      
 两个值只要有一个true就会返回true

 两个值只有都是true才会返回true

 非boolean的与或运算

 对于非boolean值 会将其先转换为boolean再运算

        var result = 1 && 2; //2 会转换 但是不会改变原值 返回最后面那个
        console.log(result);
        result = 2 && 0; //返回0 又一个值是false 返回false
        result = NaN && 0; //返回nan 都是false 返回前面那个 短路

 第一个是true则直接返回第一个值

        result = 1 || 2;

 第一个值是false返回第二个值

        result = 0 || 2;
        result = 0 || null;

      # 关系运算符

      通过关系运算符可以比较两个值之间的大小关系

      如果关系成立它会返回true, 如果关系不成立则返回false

      # >号

       判断符号左侧的值是否大于右侧的

 如果关系成立，返回true, 如果关系不成立则返回false

        console.log(5 > 10)
        var result = 5 > 10;
        console.log(result) //false
        result = 5 < 10;
        console.log(result) //true
        result = 5 >= 10;
        console.log(result) //false
        result = 5 <= 10;
        console.log(result) //true

 非数值的情况

 对于非数值

        coonsole.log(1 > true) //false
        console.log(1 >= true) //true  把true当成1
        console.log(1 > '0') //true

 任何值和NAN比较返回都是false

        console.log(10 > 'hello') //false
        console.log(1 > null) //true
        console.log(1 > undefined) //true
        console.log(true > false) //true  1>0

 如果两边都是字符串 不会将其转换为数字进行比较

 而会比较unicode编码 自行百度unicode编码表

 比较字符编码时候 时候会一位一位比较

 如果两位一样 则会比较下一位，所以借用它来对我们英文进行排序

        console.log('1' < '5') //true
        console.log('11' < '5') //true

 注意在比较两个字符串型的数字时候，一定一定要注意转型

        console.log('122345646' < +'5')

      # 相等运算符

      相等运算符 用来比较两个值是否相等

       如果相等会返回true 否则返回false

 使用==来做相等运算 

 如果只得类型不同，则会自动进行类型转换，将其转换为相同的类型 然后会比较

 == 

        console.log(1 == 1) //true
        console.log('1' == 1) // true

 NAN不和任何值相等，包括它自己本身

 可以使用isNAN()函数来判断一个值是否是NAN

        var isnan = NAN;
        console.log(isnan == NAN) //false
        console.log(isNaN(isnan)); //true

 != 不相等判断两个值是否不相等 不相等返回true

 与==相反

 ===全等 不会做自动类型转换

 ！==不全等

      # 条件运算符 也叫三元运算符

      • 语法： 条件表达式? 语句1：语句2;
      
 条件运算符，首先对条件表单时进行求值。

 如果该值为true, 则执行语句1，并返回执行结果

 如果该值为false，则执行语句2，并返回执行结果

        console.log(1 > 0 ? console.log('aaa') : console.log('bbb'));

 嵌套写法

        console.log(1 > 0 ? (1 < 0 ? alert('aaa') : alert('bbb')) : (1 > 0 ? console.log('ccc') : console.log('ddd')));
