              # 数组（Array）

              • 数组也是一个对象

              • 它和普通的对象类似，也是用来存储一些值的

              • 不同的是普通对象是使用字符串作为属性的

              • 数组则使用数字来作为索引操作元素

              • 索引：

                从0开始的整数就是索引

              • 数组的存储性能比普通对象要好，在开发中我们经常使用数组来存储一些数据

              function log(str) {
console.log(str)
}

              # 获取数组的长度

              • 可以使用length来获取数组的长度
              • 语法：数组.length
              • 对于连续的数组可以使用length获取数组的长度
              • 对于非连续的数组，使用length会获取到最大的索引
              • 尽量不要创建非连续的数组
              arr[10] = 31
log(arr)
log(arr.length)

//  修改length
 //  如果修改的length大于原长度，则多出的部分会空出来
 //  如果修改的length小于原长度，则多出来的元素会被删除
 //  为数组最后一位加值
arr[arr.length] = 50s

              # 数组的四个方法

              • push
              • 该方法可以向数组中添加一个或多个元素，并返回数组新的长度
              • 将要添加的元素作为方法的参数传递
              • 这样这个元素将会自动添加到数组的末尾
              var arr4 = [1, 2, 3]
arr4.push(4)
log(arr4) 1234
arr4.push(5, 6, 7)
log(arr4)  1234567

              # slice和splice

              var arr5 = [1, 2, 3, 4, 5]

              # slice

               不会改变原数组，返回一个新数组
 可以在数组中提取指定元素
 语法：arr.slice(start, end)
 不会包含结束索引

arr5.slice(1, 2) 2 

 第二个参数可以省略不写

arr5.slice(1) 2345

              # splice

              删除元素并向数组添加新的元素 使用splice会意向原数组，并将指定元素删除

              • 第一个参数是从哪里删除
              • 第二个参数是删除的个数
              • 第三个参数及以后，替换指定位置的元素
              • 语法：splice(起始位置，删除个数)
              • arr5.splice(0, 2) 12

              # sort

              • 数组排序
              • 根据unicode编码顺序
              • 我们可以添加一个回调函数，来指定排序规则
              • 浏览器会根据回到函数的返回值来决定元素的位置
              • 如果返回一个大于0的值，对元素会交换位置
              • 如果返回一个小于0的值，则元素位置不变
              • 如果返回一个0，则认为两个值相等，元素位置也不变
              • 如果需要升序排列 返回a-b
              • 如果需要降序排列，返回b-a
              var arr11 = arr7.sort()
log(arr11) 123456789

var arr12 = arr7.sort(function (a, b) {
 if (a > b) {

     return 1

 } else if (a < b) {

     return -1

 } else {

     return 0

 }
 return a - b 升序
return b - a 降序
})
log(arr12)
